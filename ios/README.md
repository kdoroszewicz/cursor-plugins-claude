# iOS plugin

iOS development: Swift, SwiftUI, architecture, and testing.

## Installation

```bash
agent install ios
```

## Components

### Skills

| Skill | Description |
|:------|:------------|
| `build-ios-feature` | Implement iOS features with clear architecture, state handling, and test coverage |
| `debug-ios-issue` | Reproduce and debug iOS issues with state, lifecycle, and networking diagnostics |
