# go-api-development

Go API development with standard library (1.22+) rules by Marvin Kaunda (cursor.directory)

## Installation

```bash
agent install go-api-development
```

## What's Included

### Rules

| Rule | Description | Applies To |
|:-----|:------------|:-----------|
| `go_api_development` | Go API development using the standard library net/http package and ServeMux | `**/*.go, **/go.mod` |
