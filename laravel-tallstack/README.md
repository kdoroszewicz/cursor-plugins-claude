# laravel-tallstack

Laravel TALL Stack (Tailwind, Alpine.js, Livewire, Laravel) with DaisyUI rules by Ismael Fi (cursor.directory)

## Installation

```bash
agent install laravel-tallstack
```

## What's Included

### Rules

| Rule | Description | Applies To |
|:-----|:------------|:-----------|
| `laravel_tallstack` | Laravel TALL Stack development with Livewire, Alpine.js, TailwindCSS, and DaisyUI | `**/*.php, **/*.blade.php, **/*.js` |
