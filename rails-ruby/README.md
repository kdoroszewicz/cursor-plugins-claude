# rails-ruby

Ruby on Rails, PostgreSQL, Hotwire, and Tailwind development rules by Theo Vararu (cursor.directory)

## Installation

```bash
agent install rails-ruby
```

## What's Included

### Rules

| Rule | Description | Applies To |
|:-----|:------------|:-----------|
| `rails_ruby` | Ruby on Rails development with Hotwire, Turbo, Stimulus, and Tailwind CSS | `**/*.rb, **/*.erb, **/Gemfile` |
