# Design plugin

Product design: UX specs, design systems, handoff, and iteration.

## Installation

```bash
agent install design
```

## Components

### Skills

| Skill | Description |
|:------|:------------|
| `create-design-spec` | Turn product requirements into a clear design spec and handoff checklist |
| `review-design-quality` | Evaluate a design solution for usability, consistency, and implementation readiness |
