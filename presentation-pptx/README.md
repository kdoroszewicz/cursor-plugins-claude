# Presentation PPTX plugin

Presentation authoring: narrative design, slide outlines, and PPTX-ready content.

## Installation

```bash
agent install presentation-pptx
```

## Components

### Skills

| Skill | Description |
|:------|:------------|
| `build-presentation-outline` | Convert goals into a compelling slide narrative with speaker-ready structure |
| `improve-slide-deck` | Critique and improve decks for clarity, narrative flow, and audience impact |
